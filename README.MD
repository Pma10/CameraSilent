# CameraSilent
> 갤럭시 카메라 무음 세팅 도구<br>
> Galaxy Camera Silent Setting Tool

이 도구는 ADB를 활용하여 갤럭시 스마트폰의 카메라 셔터음을 무음으로 설정합니다.

## 주요 기능
- **USB 연결 방식**: 유선 케이블을 통한 전통적인 방식
- **무선 디버깅 방식**: 안드로이드 11 이상 기기에서 같은 Wi-Fi를 사용하여 선 없이 연결 (Wireless ADB)
- **자동 ADB 관리**: 필요한 구동 파일을 자동으로 다운로드하고 설정

## 사용법
1. 휴대폰의 **개발자 옵션**을 활성화합니다.
2. `main.py`를 실행합니다.
3. 화면의 안내에 따라 연결 방식을 선택합니다:
   - **USB**: 케이블 연결 후 진행
   - **무선**: '무선 디버깅' 활성화 후 페어링 코드로 연결
4. 설정이 완료되면 카메라 앱에서 셔터음이 나지 않습니다. (시스템 소리/진동 모드 확인 필요)

## 주의사항
- 안드로이드 버전에 따라 작동 여부가 다를 수 있습니다.
- 시스템 업데이트 후 초기화될 수 있으니 그럴 경우 다시 실행해 주세요.
